This application is a home made console for WildFly. Don't expect too much, it's basically a proof of concept. The source code is available on github (the name of the repository will probably change soon).

The current version works with CORS but needs some additional setup :

  • Disable the authentication of the WildFly Management interface. Because, I don't know to make it work.
  • Use a reverse proxy between the browser and WildFly Management interface. Because WildFly does not support CORS, in version 8.x.

For example, you can setup the mod_proxy module in Apache Httpd 2.4 : 

Listen 90
<VirtualHost *:90>

    ProxyRequests Off
    RequestHeader unset Origin

    RewriteEngine on
    RewriteCond %{REQUEST_METHOD} !OPTIONS
    RewriteRule ^/management$ http://localhost:9990/management [P]
    RewriteRule ^/management/(.*)$ http://localhost:9990/management/$1 [P]
    RewriteCond %{REQUEST_METHOD} OPTIONS
    RewriteRule ^(.*)$ $1 [R=200,L]
    ProxyPassReverse /management http://localhost:9990/management

    Header always set Access-Control-Allow-Origin: "*"
    Header always set Access-Control-Allow-Methods "POST, OPTIONS"
    Header always set Access-Control-Allow-Headers "content-type"

</VirtualHost>
