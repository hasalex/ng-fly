This application is a home made console for WildFly. Don't expect too much, it's basically a proof of concept. The source code is available on github.

It works with CORS but needs some additional setup.

  • With WildFly 8.0 or 8.1, it doesn't work. Move to a more recent version.
  • The WildFly 8.2 management doesn't support CORS. The workaround is to use a reverse proxy between the browser and WildFly Management interface. See the documentation for an example of Apache httpd configuration.
  • With WildFly 9 or further, you just have to enable CORS on the management interface.

How to enable CORS with jboss-cli : 

cd /core-service=management/management-interface=http-interface
:write-attribute(name=allowed-origins, value=["http://localhost:8888"])
/:reload
